À propos de TaskBoard

TaskBoard est un petit projet fictif de tableau de tâches, construit uniquement avec HTML et CSS — aucun JavaScript — pour que la formation reste accessible à un public débutant, sans notion de programmation préalable.

Le projet est volontairement statique : les cartes de tâches sont écrites directement dans le HTML, et le formulaire de la page d'accueil ne déclenche aucune action. L'objectif est de pratiquer GitHub Copilot sur du HTML et du CSS réalistes, organisés comme un vrai petit projet (plusieurs pages, plusieurs fichiers de styles).

Consultez docs/architecture.md pour le détail de l'organisation des fichiers.
